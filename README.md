## üìñ Quick Start Guide

### Basic Workflow

1. **Upload Base Image**
   - Click "Choose File" under "1. Upload Base Image"
   - OR select a background from the dropdown

2. **Add Text/Emojis (Optional)**
   - Enter title text
   - Add numbers/subtitle
   - Select emoji or shape from dropdown
   - Adjust size, color, position, and effects

3. **Create Preview**
   - Click **"Create Flow Image"**
   - Review your design
   - Make adjustments and click again to update

4. **Keep as Base (Optional)**
   - Click **"KEEP"** to lock in your current design
   - This allows you to add more layers

5. **Add Second Image (Optional)**
   - Upload a second image
   - Click the thumbnail to toggle views
   - Edit, erase, or apply effects
   - Click **"Save Edits"** when done

6. **Download**
   - Click **"Download Final Cover"**
   - Find your cover in: `Pictures/CoverMaker/`

### Advanced: Using Second Image Features

1. **Toggle Between Images**
   - Click the small preview thumbnail to switch views
   - Main image: Add text and position
   - Second image: Edit, erase, apply effects

2. **Drag & Drop Overlay**
   - While viewing main image, drag the second image thumbnail onto the preview
   - Position with mouse
   - Adjust scale and rotation in controls panel
   - Click **"Apply Position"** to lock

3. **Eraser Tool**
   - Switch to second image view
   - Use slider to adjust brush size
   - Click and drag to erase
   - Click **"Save Edits"** when finished

4. **Shape Masks**
   - Select a shape from dropdown
   - Adjust offset X/Y for positioning
   - Click **"Apply Shape Mask"**
   - Use Pan controls to reposition image inside shape

5. **3D Wrap**
   - Select 3D shape (sphere, cube, etc.)
   - Adjust rotation and zoom
   - Click **"Apply 3D Wrap"**
   - Toggle back to main image to see result
